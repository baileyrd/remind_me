  <name>Task 1: Create tests/__init__.py and conftest.py with all shared fixtures</name>
  <files>tests/__init__.py, tests/conftest.py</files>
  <action>
Create `tests/__init__.py` as an empty file (package marker).

Create `tests/conftest.py` with the following fixtures:

1. **`tmp_memory_dir` (session-scoped autouse fixture)**:
   - Use `tmp_path_factory` to create a temporary directory for the test session
   - Monkeypatch `remind_me_mcp.config.MEMORY_DIR`, `remind_me_mcp.config.DB_PATH`, `remind_me_mcp.config.PID_FILE`, and `remind_me_mcp.config.IMPORT_LOG` to point at the temp dir
   - This ensures no test ever touches the real `~/.remind-me/` directory
   - IMPORTANT: Also monkeypatch the same attributes on any module that imports them directly (e.g., `remind_me_mcp.db.DB_PATH`, `remind_me_mcp.pid.DB_PATH`, `remind_me_mcp.pid.PID_FILE`, `remind_me_mcp.server.DB_PATH`, `remind_me_mcp.api.DB_PATH`)

2. **`db_conn` (function-scoped fixture)**:
   - Create a fresh `sqlite3.Connection` using `":memory:"` (NOT a file)
   - Set `row_factory = sqlite3.Row`
   - Call `_ensure_schema(db)` from `remind_me_mcp.db` to create the full schema (memories table, chat_imports table, FTS5 virtual table, all triggers, all indexes)
   - Do NOT attempt to create `memories_vec` table (no sqlite-vec in tests) — `_ensure_schema` already handles this gracefully
   - Monkeypatch `remind_me_mcp.db._get_db` to return this in-memory connection (so all code calling `_get_db()` gets the test DB)
   - Also monkeypatch `remind_me_mcp.api._get_db` since api.py imports `_get_db` directly
   - Yield the connection, then close it in teardown

3. **`mock_embedder` (function-scoped fixture)**:
   - Create a class `FakeEmbedder` that mimics `_Embedder` interface:
     - `embed(texts: list[str]) -> np.ndarray` — returns a deterministic (len(texts), 384) float32 array. Use `np.random.default_rng(seed=42)` seeded on a hash of each text so same text always produces same vector. L2-normalize the output.
     - `embed_one(text: str) -> bytes` — calls `embed([text])[0].tobytes()`
     - `available` property returns `True`
   - Monkeypatch `remind_me_mcp.db._get_embedder` to return the FakeEmbedder instance
   - Monkeypatch `remind_me_mcp.embeddings._get_embedder` to return it
   - Monkeypatch `remind_me_mcp.embeddings._embedder` to the instance
   - Return the FakeEmbedder instance so tests can inspect calls

4. **`memory_factory` (function-scoped fixture, depends on db_conn)**:
   - Return a callable `create_memory(**overrides) -> dict` that:
     - Accepts keyword overrides for any memory field
     - Defaults: `content="Test memory content"`, `category="general"`, `tags=[]`, `source="manual"`, `metadata={}`, auto-generated `id` via `_make_id(content)`, auto-generated `created_at` and `updated_at` via `_now_iso()`
     - Inserts the row into the in-memory database via `db_conn.execute(INSERT INTO memories ...)`
     - Commits the insert
     - Returns the full memory dict (all fields including id, created_at, updated_at)
   - This avoids repeating INSERT boilerplate in every test

5. **`sample_chat_json` (function-scoped fixture)**:
   - Use `tmp_path` to create a temporary JSON file containing a sample Claude export:
     ```json
     {"chat_messages": [
       {"sender": "human", "content": [{"type": "text", "text": "What is Python?"}]},
       {"sender": "assistant", "content": [{"type": "text", "text": "Python is a programming language."}]}
     ]}
     ```
   - Return the `Path` object pointing to the file

6. **`sample_chat_md` (function-scoped fixture)**:
   - Use `tmp_path` to create a temporary Markdown file containing:
     ```
     ## Human
     What is Python?

     ## Assistant
     Python is a programming language.
     ```
   - Return the `Path` object

Use `from __future__ import annotations` at the top. Import `pytest`, `sqlite3`, `json`, `numpy as np`, `Path` from pathlib. Import `_ensure_schema` from `remind_me_mcp.db`, `_make_id` and `_now_iso` from `remind_me_mcp.db`.

Do NOT import `remind_me_mcp.tools` or `remind_me_mcp.server` in conftest.py (those trigger MCP registration side effects). Tests that need those will import them locally.

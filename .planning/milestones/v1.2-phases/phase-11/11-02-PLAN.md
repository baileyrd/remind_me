**In models.py**, add:

1. `MemoryClassification` -- a simple model with fields:
   - memory_id: str (Field, min_length=1)
   - memory_type: str (Field, must be one of: decision, preference, fact, insight, learning, blocker, action_item)
   Use a field_validator to check memory_type is in the allowed set.

2. `ReclassifyInput` -- input for reclassify tool:
   - classifications: list[MemoryClassification] (Field, min_length=1, max_length=100)
   - model_config with extra="forbid"

3. `ReclassifyBatchInput` -- input for batch fetch tool:
   - batch_size: int = Field(default=20, ge=1, le=100, description="Number of unclassified memories to return")
   - model_config with extra="forbid"

Add both to __all__.

**In tools.py**, add:

1. Import DECAY_RATES from remind_me_mcp.vitality
2. Import ReclassifyInput, ReclassifyBatchInput from models

3. `remind_me_reclassify` tool (name="remind_me_reclassify"):
   - annotations: readOnlyHint=False, destructiveHint=False, idempotentHint=True
   - For each classification in params.classifications:
     - Fetch memory by ID
     - If not found, add to not_found list
     - Set memory_type = classification.memory_type
     - Set decay_rate = DECAY_RATES.get(classification.memory_type, 0.10)
     - UPDATE memories SET memory_type=?, decay_rate=?, updated_at=? WHERE id=?
   - db.commit()
   - Return JSON: {"updated": N, "not_found": [...], "total": len(classifications)}

4. `remind_me_reclassify_batch` tool (name="remind_me_reclassify_batch"):
   - annotations: readOnlyHint=True, destructiveHint=False, idempotentHint=True
   - SELECT id, substr(content, 1, 500) as content_snippet, category, tags FROM memories WHERE memory_type = 'unclassified' LIMIT ?
   - Return JSON array of {id, content_snippet, category, tags} for Claude to classify
   - Include total_unclassified count in response

**In tests/test_tools.py**, add integration tests using the existing test fixture pattern (temp DB with test memories). Create test memories with memory_type='unclassified', then test both tools.

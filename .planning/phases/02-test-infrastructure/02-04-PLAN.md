  <action>
Create `tests/test_api.py`.

Import setup:
```python
from __future__ import annotations
import json
import pytest
from starlette.testclient import TestClient
from remind_me_mcp.api import _build_api_app
```

Create a `client` fixture:
```python
@pytest.fixture
def client(db_conn):
    """Build a Starlette TestClient with the in-memory db monkeypatched."""
    app = _build_api_app()
    return TestClient(app)
```

The `db_conn` fixture from conftest already monkeypatches `remind_me_mcp.api._get_db` to return the in-memory connection, so all API route handlers will use the test database.

**Dashboard / index tests:**
- `test_dashboard_serves_html(client)` — `GET /`. Assert status 200. Assert `"text/html"` in `response.headers["content-type"]`. Assert `"Remind Me"` in response text (page title). Assert `"react"` in response text (React script tag).

**GET /api/stats tests:**
- `test_api_stats_empty(client)` — No memories in db. `GET /api/stats`. Assert 200. Assert JSON body has `"total": 0`.
- `test_api_stats_with_data(client, memory_factory)` — Create 3 memories with different categories. `GET /api/stats`. Assert `total == 3`. Assert categories dict contains expected category names.

**GET /api/memories (list) tests:**
- `test_api_list_empty(client)` — No memories. `GET /api/memories`. Assert 200. Assert `"total": 0` and empty `"memories"` list.
- `test_api_list_with_data(client, memory_factory)` — Create 3 memories. `GET /api/memories`. Assert `total >= 3` and `count == 3`.
- `test_api_list_filter_by_category(client, memory_factory)` — Create memories in categories "work" and "personal". `GET /api/memories?category=work`. Assert all returned have `category == "work"`.
- `test_api_list_pagination(client, memory_factory)` — Create 5 memories. `GET /api/memories?limit=2&offset=0`. Assert `count == 2` and `has_more == True`. `GET /api/memories?limit=2&offset=4`. Assert `has_more == False`.

**POST /api/memories (add) tests:**
- `test_api_add_memory(client)` — POST with JSON `{"content": "Test memory", "category": "test"}`. Assert status 201. Assert response JSON has `id`, `content`, `category`.
- `test_api_add_missing_content(client)` — POST with `{"category": "test"}` (no content). Assert status 400. Assert error message mentions "content".
- `test_api_add_invalid_json(client)` — POST with non-JSON body. Assert status 400.

**GET /api/memories/{id} tests:**
- `test_api_get_memory(client, memory_factory)` — Create a memory, `GET /api/memories/{id}`. Assert 200 and correct content.
- `test_api_get_not_found(client)` — `GET /api/memories/nonexistent`. Assert 404.

**PUT /api/memories/{id} (update) tests:**
- `test_api_update_content(client, memory_factory)` — Create a memory. `PUT /api/memories/{id}` with `{"content": "Updated content"}`. Assert 200. Verify content changed via GET.
- `test_api_update_tags(client, memory_factory)` — Update tags. Verify tags changed.
- `test_api_update_not_found(client)` — `PUT /api/memories/nonexistent` with `{"content": "x"}`. Assert 404.
- `test_api_update_no_fields(client, memory_factory)` — `PUT /api/memories/{id}` with `{}`. Assert 400 ("No fields to update").

**DELETE /api/memories/{id} tests:**
- `test_api_delete_memory(client, memory_factory)` — Create a memory. `DELETE /api/memories/{id}`. Assert 200. Verify `GET /api/memories/{id}` returns 404.
- `test_api_delete_not_found(client)` — `DELETE /api/memories/nonexistent`. Assert 404.

**GET /api/memories/search tests:**
- `test_api_search_finds_match(client, memory_factory)` — Create a memory with content "Python asyncio patterns". `GET /api/memories/search?q=asyncio`. Assert 200 and `count >= 1`.
- `test_api_search_no_query(client)` — `GET /api/memories/search` (no q param). Assert 400 with error about missing "q".
- `test_api_search_no_results(client)` — `GET /api/memories/search?q=zzzznotfound`. Assert 200 and `count == 0`.

**POST /api/import tests:**
- `test_api_import_file(client, db_conn, sample_chat_json)` — POST `{"file_path": str(sample_chat_json)}`. Assert 200 and response contains `"status": "ok"` with `memories_created >= 1`.
- `test_api_import_missing_path(client)` — POST `{"file_path": ""}`. Assert 400.
- `test_api_import_nonexistent_file(client)` — POST `{"file_path": "/nonexistent/file.json"}`. Assert 400 with "not found".

**REST CRUD cycle test:**
- `test_api_crud_cycle(client)` — POST add -> GET verify -> PUT update -> GET verify change -> DELETE -> GET 404. Full round-trip via HTTP API.

All tests are synchronous (Starlette TestClient handles async routes internally). Tests use `db_conn` and `memory_factory` fixtures from conftest.

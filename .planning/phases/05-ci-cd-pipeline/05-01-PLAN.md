<task type="auto">
  <name>Task 1: Create GitHub Actions CI workflow</name>
  <files>.github/workflows/ci.yml</files>
  <action>
Create the directory `.github/workflows/` and the file `.github/workflows/ci.yml` with the following configuration:

**Workflow name:** `CI`

**Triggers:** `on: push` (all branches) and `on: pull_request` (all branches). Both triggers required by CICD-01.

**Job:** Single job named `ci` with display name `Python ${{ matrix.python-version }}`.

**Matrix strategy:**
- `python-version: ["3.11", "3.12"]` (CICD-01 requirement)
- `fail-fast: false` (both versions always report independently)

**Runner:** `ubuntu-latest`

**Steps (in order):**
1. `actions/checkout@v4` — check out repository
2. `astral-sh/setup-uv@v5` with `python-version: ${{ matrix.python-version }}` — install uv and the correct Python
3. **Install dependencies:** `uv pip install -e ".[semantic]" pytest-cov ruff pytest-asyncio`
   - `[semantic]` extras required for onnxruntime/sqlite-vec imports in test collection
   - `pytest-asyncio` required because `asyncio_mode = "auto"` is set in pyproject.toml but pytest-asyncio is NOT a declared dependency
   - `pytest-cov` for coverage measurement and threshold enforcement
   - `ruff` for lint (may already be installed but explicit is better)
4. **Lint:** `ruff check --output-format=github .`
   - `--output-format=github` produces inline PR annotations on lint errors
5. **Test with coverage:** `pytest --cov=remind_me_mcp --cov-report=term-missing --cov-fail-under=74`
   - `--cov=remind_me_mcp` scopes coverage to the package (not test files)
   - `--cov-report=term-missing` shows which lines are missed in the log
   - `--cov-fail-under=74` sets the gate at measured baseline (76%) minus 2% headroom per STATE.md guidance. CICD-02 requirement specifies 80% as the goal, but current coverage is 76% — setting at 80% would cause every CI run to fail. Document this gap with a YAML comment.

Add a YAML comment at the top of the file noting the coverage threshold rationale:
```yaml
# Coverage gate: 74% (measured 76% minus 2% headroom per STATE.md).
# Target is 80% (CICD-02) — will increase as tests are added in Phases 6-8.
```
